Tipps für SetlX in LON-CAPA

Verwenden Sie keine Umlaute.

Tipps für SetlX auf Windows-Computern

  • Wenn load() nicht funktioniert: Im Pfad benötigen Sie eventuell jeweils doppelte Schrägstriche (C:\\Users\\Public\\setlx\\truthtable.stlx). Außerdem sollten Sie darauf achten, ob die Datei eventuell eine nicht-sichtbare Endung ".txt" hat.
  • Wenn Copy/Paste nicht funktioniert: Haben Sie multiLineMode(true); ausgeführt?

Tipps für SetlX auf Mac-Computern

Öffnen Sie ein Terminal-Fenster (in Programme/Dienstprogramme). Tippen Sie in das Terminal-Fenster cd gefolgt von dem Pfad zu Ihrem setlX-Ordner. Also z.B. cd Desktop/setlx. Danach führen Sie setlX aus mit ./setlX -m. Wenn Sie in setlX die Pfeiltasten verwenden wollen um vorhergehende Befehle ändern zu können, müssen Sie das Programm rlwrap installieren.

Wenn Sie rlwrap nicht installiert haben, ist am einfachsten, wenn Sie Ihren Code in einer Datei (z.B. als code.stlx) in dem gleichen Ordner speichern, wo setlX installiert ist. Sie können die Datei mit ./setlX -m DATEINAME, also z.B. ./setlX -m code.stlx ausführen. Bei Dateiausführung sehen Sie aber nur Ergebnisse, wenn Sie print-Befehle angeben, z.B. 5+6; zeigt nichts an. print(5+6); zeigt etwas an.

Achten Sie darauf, dass Ihnen das Betriebssytem eventuell nicht den vollen Dateinamen anzeigt. Wenn setlX Ihre Datei nicht finden kann, geben Sie ls im Terminal-Fenster in dem Ordner ein, wo Ihre Datei gespeichert ist, um zu sehen, welches der tatsächlich gespeicherte Dateiname ist. Mit mv ALTERDATEINAME NEUERDATEINAME können Sie Ihre Datei umbenennen.
