1. Mengen und Mengenoperatoren

Eine Menge in SetlX: 

{1, 2, 3};

SetlX stellt die folgenden Operatoren für Mengen bereit:

inElement in Menge
..Aufzählung
+Vereinigung
*Schnittmenge
-Differenz
#Kardinalität (Anzahl der Elemente)
==Gleich
!=Ungleich
<=Teilmenge (auch >=)
<echte Teilmenge (auch >)

Weitere Operatoren, die Sie im Verlauf des Semesters kennenlernen werden sind: >< kartesisches Produkt, ** 2 kartesisches Produkt einer Menge mit sich selbst und 2 ** Potenzmenge. Darüber hinaus gibt es in SetlX weitere Operatoren, die die Arbeit mit Mengen vereinfachen.

Neben der Angabe als Literal, können Mengen auch mittels einer Konstruktionsregel angegeben werden (set comprehension):

{ 2 * a : a in {1..10}};

Diese Notation ist in der Regel deutlich kürzer und mit etwas Übung leichter verständlich als die explizite Angabe aller Elemente.

Probieren Sie diese Operatoren selbst einmal aus. Hier einige Anregungen:

{1..100};
{47, 11} == {11, 47};
{1, 3, 5} + {2, 4, 6};
{1, 3, 5} - {2, 3, 4};
{1, 3, 3} == {1, 3};
{-5..5} * {1..100};
#{a/b : a in {1..100}, b in {1..100}};
{-5..5} <= {-5..5}*{1..100};
{1, 2, 3} < {1, 2, 3};
2 in {1, 2, 3};
{2} in {1, 2, 3};
{2} < {1, 2, 3};
2 < {1, 2, 3};

Entsprechen die Ergebnisse Ihren Erwartungen? Wie unterscheiden sich die Ergebnisse von SetlX ggf. von Ihren Erwartungen?
Können Sie weitere Beispiele finden, die nicht Ihren Erwartungen entsprechen? Nennen Sie diese.
Erklären Sie, warum die letzten vier Befehle zu unterschiedlichen Ergebnissen führen.

2. Prozeduren

Funktionen lassen sich in SetlX z.B. durch Prozeduren repräsentieren. Eine Funktion, die zu jedem Element einer Menge 1 hinzuaddiert, könnte beispielsweise so aussehen: 

addiere1 := procedure(s) {    
     return { a + 1 : a in s }; 
};
=> addiere1({1,2,3});
~< Result: {2, 3, 4} >~

Um dieses Beispiel in SetlX zu testen, können Sie die beiden Zeilenumbrüche entfernen und den gesamten Quelltext in einer Zeile eingeben oder Sie aktivieren den Mehrzeilenmodus mittels multiLineMode(true). Im zweiten Fall müssen Sie nun die Return Taste zweimal betätigen, um die Zeilen auszuführen.

Schreiben Sie eine Prozedur cSumme, die zwei Mengen als Eingabe akzeptiert, jedes Element der einen Menge und jedes Element der anderen Menge addiert und als Ergebnis die Menge dieser Summen zurückgibt.

Beispiel für einen Aufruf von cSumme:

=> cSumme({1,2,3}, {10,20,30});
~< Result: {11, 12, 13, 21, 22, 23, 31, 32, 33} >~
