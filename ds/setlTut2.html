1. Aussagenlogische Operatoren

In SetlX stehen u.a. folgende aussagenlogische Operatoren zur Verfügung:

&&Konjunktion (logisches Und)
||Disjunktion (logisches Oder)
!Negation (logisches Nicht)
=>Implikation
<==>Äquivalenz

Diese Operatoren sind für Elemente von Zahlenmengen nicht definiert, stattdessen erwarten sie die Bool’schen Werten true und false (wahr und falsch).

Probieren Sie diese Operatoren selbst einmal aus. Hier einige Anregungen:

true;
true && false;
true || false;
true && !false;
true => false;
true <==> false;
true && (true || false);
!(true && false);

Entsprechen die Ergebnisse Ihren Erwartungen? Wie unterscheiden sich die Ergebnisse von SetlX ggf. von Ihren Erwartungen?
Können Sie weitere Beispiele finden, die nicht Ihren Erwartungen entsprechen? Nennen Sie diese.
Stellen Sie mit Hilfe von SetlX die Wahrheitstabelle für a => b für alle möglichen Belegungen von a und b auf.

2. Vergleichsoperatoren

Vergleichsoperatoren können Wahrheitswerte vergleichen.

Diese Operatoren liefern dann als Ergebnis einen Wahrheitswert, also entweder true oder false. Das bedeutet unter anderem, dass die Ergebnisse mittels der im vorherigen Abschnitt eingeführten aussagenlogischen Operatoren verkettet werden können.

Probieren Sie diese Operatoren selbst einmal aus. Hier einige Anregungen:

47 + 11 == 4711 && 2**3 < 3**2;
2**3 < 3**2 == 47 + 11 == 58; 
(2**3 < 3**2) == (47 + 11 == 58);

Entsprechen die Ergebnisse Ihren Erwartungen? Wie unterscheiden sich die Ergebnisse von SetlX ggf. von Ihren Erwartungen?
Können Sie weitere Beispiele finden, die nicht Ihren Erwartungen entsprechen? Nennen Sie diese.
Erklären Sie, warum die letzten beiden Befehle zu unterschiedlichen Ergebnissen führen.
