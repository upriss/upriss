SetlX-Tutorial Teil 1

Dieses Tutorial soll Sie mit der Verwendung von SetlX vertraut machen. Nach jedem Abschnitt finden Sie einige Beispiele zum Ausprobieren. Diese Beispiele sind in der Regel möglichst einfach gehalten. Es geht hierbei nicht um die richtigen Ergebnisse, sondern vielmehr darum, dass Sie Ihre eigenen Erwartungen überprüfen.

Verhält sich das System so, wie Sie es vor der Ausführung eines Befehls vorhergesagt hätten?

Nachdem Sie das Tutorial durchgearbeitet haben sollen folgende Punkte erfüllt sein:

  • Sie haben eine funktionierende Installation von SetlX auf Ihrem Rechner, die auch während der Vorlesung einsatzbereit ist.
  • Sie können die gängigen Operationen benennen, beschreiben und in SetlX umsetzen.

Im Verlauf des Semesters werden Sie sich vertiefend mit SetlX und der diskreten Mathematik beschäftigen.

Download und Installation

Laden Sie SetlX von der Webseite http://www.randoom.org/Software/SetlX herunter und installieren Sie das Programm. Hinweise zur Installation finden Sie auf der genannten Seite. Auf der Webseite finden Sie auch einen Link zu dem offiziellen Tutorial mit weiterführenden Informationen.

1. Arithmetische Operatoren

Starten Sie SetlX. Auf Ihrem Bildschirm erscheint nach dem Begrüßungstext der SetlX-Prompt "=>".

Sie können SetlX jederzeit durch Eingabe von exit; verlassen. Die meisten Befehle in SetlX (Ausdrücke und Zuweisungen) werden durch ein Semikolon abgeschlossen (z.B. 2+2;). SetlX stellt die folgenden arithmetischen Operatoren bereit:

+Addition
-Subtraktion
*Multiplikation
**Potenz
/Division
\ganzzahlige Division
%Modulus

Probieren Sie diese Operatoren selbst einmal aus. Hier einige Anregungen: 

47 + 11;
2/3 + 7/5;
2*3 + 7*5;
2\3 + 7\5;
2**8;
2**3**5;
2*(3+4)**2;
2.0/3 + 7/5;
47 % 11;

2. Vergleichsoperatoren

Sie können Aussagen miteinander vergleichen. Zum Beispiel "2 hoch 3 ist kleiner als 3 hoch 2" oder "die Summe aus 47 und 11 ist gleich dem Produkt aus 29 und 2". Um derartige Aussagen in SetlX formulieren zu können, stehen folgende Operatoren zur Verfügung:

==Gleich (vgl. <==>)
!=Ungleich
>Größer
>=Größer gleich
<Kleiner
<=Kleiner gleich

Probieren Sie diese Operatoren selbst einmal aus. Hier einige Anregungen:

2**3 < 3**2;
47 + 11 == 58;
47 + 11 == 29 * 2;
47 + 11 == 4711;
"Hello World" != "hello World";

Entsprechen die Ergebnisse Ihren Erwartungen? Wie unterscheiden sich die Ergebnisse von SetlX ggf. von Ihren Erwartungen?

3. Zuweisung und Variablen

Sie können Variablen verwenden, um z.B. bestimmte Werte oder Ergebnisse aus Berechnungen für die spätere Verwendung zu speichern. Um Variablen Werte zuzuweisen, stellt SetlX den Zuweisungsoperator := bereit. Variablennamen in SetlX beginnen immer mit einem Kleinbuchstaben, auf den Buchstaben, Zahlen oder Unterstriche folgen können. Beispiel:

a := 23;
a := a - 20;
b:= 4/5;
c:= b - a;
text_1:="Hello World";

5. Dateien laden

Sie können Ihre Programme auch mit einem Editor ihrer Wahl erstellen und bearbeiten. Dafür legen Sie eine Textdatei an, in die Sie die Befehle zeilenweise eintragen. Diese Datei können Sie dann mittels load("dateipfad") in SetlX laden.

Speichern Sie die folgenden Zeilen in einer Datei (z.B. hello.stlx), laden Sie die Datei in SetlX und führen Sie danach die Prozedur hallo() aus. 

hallo := procedure(){
     print("Hallo!\nSie haben die Datei erfolgreich geladen!");
};

Sie sollten nun folgende Ausgabe sehen:

Hallo!
Sie haben die Datei erfolgreich geladen!
~< Result: om >~
