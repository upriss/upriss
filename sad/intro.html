CO32034 Server Administration, Spring 2005

This module covers:

  • Basic Unix
  • Linux administration
  • MySQL database administration
  • Apache Web Server administration
  • Non-Linux OS

Delivery

  • 1 hour lecture
  • 2 hour tutorial

Virtual Machines

The tutorials involve using a simulated Linux server. You will be administering your own server!

Assessment

  • The coursework/exam weighting is 50/50.
  • The coursework instructions will be released in week 4 or 5. The coursework is due at the end of week 9.
  • The exam consists of 20 short-answer questions. (Note: the resit exam consists of 3 essay questions (to be selected out of 5 questions).)
